AI-powered coaching dashboard analyzing 4,343 corners across 7 tracks to identify driver improvement opportunities.

## Features
- **AI-Powered Coaching**: Top 3 improvement opportunities per driver
- **Predictive Modeling**: Random Forest model predicts lap time improvements
- **Cross-Track Analysis**: Driver performance across 7 championship circuits
- **Multi-Driver Comparison**: Side-by-side driver analysis

## Quick Start
```
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack
- **Python 3.9+**
- **Streamlit** - Interactive dashboard
- **Scikit-learn** - ML models
- **Plotly** - Visualizations
- **Pandas/NumPy** - Data processing

## Dataset
- 4,343 corners analyzed
- 64 drivers tracked
- 7 circuits (VIR, Road America, COTA, Sebring, Sonoma, Barber, Indianapolis)

Built for Toyota GR Cup Championship Analytics Challenge
